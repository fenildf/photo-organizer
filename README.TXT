Photo Organizer Script
----------------------

   The script maintains all your photos in a pre-determined structure and makes downloading and organizing of new photos easy. 

Usage
-----
  
   Just run photo-store.py

   The script will ask for your existing photo repository. If there are no files in this repository, it will create a new one.
